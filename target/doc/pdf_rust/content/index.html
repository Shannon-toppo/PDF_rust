コンテントストリーム（ページ記述）の解析と生成。

コンテントストリームは「オペランド列 + 演算子」の繰り返しで構成される （§7.8.2）。本モジュールはこれを Operation の列に分解する。 インライン画像（BI ... ID <バイナリ> EI）はデータ部をスキップして 1 つの BI 演算として返す。
