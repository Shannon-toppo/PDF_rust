相互参照テーブル（クロスリファレンス）の読み取り。

以下の 3 形式すべてに対応する:

  • 古典的な xref テーブル（§7.5.4）
  • クロスリファレンスストリーム（§7.5.8, PDF 1.5+）
  • ハイブリッド参照ファイル（/XRefStm）

また、xref が壊れている場合はファイル全体を走査して n g obj を拾い集める再構築（reconstruct）を行う。
